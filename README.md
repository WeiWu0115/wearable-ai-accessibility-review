# Cognitive Accessibility for Wearable Intelligence

This repository hosts a scoping review project website and meeting brief for:

**Cognitive Accessibility for Wearable Intelligence: A Scoping Review of Neurodiversity, Health-Based Support, and Human-AI Interaction Beyond the Screen**

The project maps health, rehabilitation, neurodiversity, HCI, XR, wearable computing, and human-AI interaction literature toward a new accessibility framework for wearable intelligence.

## Website Files

- `index.html`: GitHub Pages homepage
- `search_protocol.html`: CHI-level scoping review search protocol
- `meeting_brief.html`: full meeting brief webpage
- `meeting_brief.pdf`: PDF version of the brief
- `meeting_brief.md`: Markdown source

## Working Framework

**CAWI: Cognitive Accessibility for Wearable Intelligence**

Initial principles:

1. Contextual Perceivability
2. Sensory Modulation
3. Cognitive Legibility
4. Body-Capability-Aware Operability
5. Attention Respect
6. Adaptive Transparency
7. Inference Accountability

## Review Goal

The review asks how health-based support strategies for cognitive, sensory, motor, perceptual, emotional, environmental, and AI-mediated access needs can be translated into HCI and UI/UX accessibility principles for wearable AI systems.
