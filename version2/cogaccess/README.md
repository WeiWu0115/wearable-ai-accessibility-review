## 三句话结论
1. 存在以 **W3C COGA 八目标**为骨架的跨人群共性原则（清晰语言·降记忆·降负荷/专注·可预测·防错反馈·帮助·可定制·多模态·参与式）。
2. **自闭症**特异性最强：**感官调节 + 抗界面变化 + 字面化语言 + 深度定制**；与 ADHD/TBI 在"专注+定制"高度共振。
3. 空白：跨人群对照验证、成人/职场情境、AI 时代新设计责任。

## 重要局限
ACM(403)/IEEE(418) 反爬，**全文未逐篇通读**；原则提取基于题录+摘要+开放镜像（W3C COGA、GAIA、arXiv/PMC）。具体条目计数（GAIA 28 条等）来自二手描述，建议正式投稿前逐篇核对原文 PDF。
