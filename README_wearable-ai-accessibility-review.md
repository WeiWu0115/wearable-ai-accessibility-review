# Wearable AI Accessibility Review 项目包

日期：2026-05-27

这是 “Cognitive Accessibility for Wearable Intelligence” scoping review 项目的本地备份包。

## 最重要的入口

- 在线网页：<https://weiwu0115.github.io/wearable-ai-accessibility-review/>
- GitHub 仓库：<https://github.com/WeiWu0115/wearable-ai-accessibility-review>
- 当前版本：`v0.2-chi-search-protocol-2026-05-27`

## 本地文件怎么读

打开项目文件夹后，优先看这些文件：

1. `index.html`  
   项目网页首页。双击可以在浏览器里打开。

2. `search_protocol.html`  
   CHI-level scoping review 搜索协议，包括 7 层 search strategy、数据库、医学 controlled vocabulary、HCI venue hand-search、screening plan。

3. `meeting_brief.pdf`  
   可直接发给合作者或开会使用的 PDF brief。

4. `meeting_brief.md`  
   会议版研究地图的 Markdown 源文件。

5. `CHANGELOG.md`  
   版本记录。今天的 baseline version 已经记录在这里。

6. `README.md`  
   GitHub 仓库说明。

## 文件夹内容

- `index.html`: GitHub Pages 首页
- `search_protocol.html`: CHI-level Search Protocol 页面
- `meeting_brief.html`: 完整 meeting brief 网页版
- `meeting_brief.pdf`: PDF 版本
- `meeting_brief.md`: Markdown 版本
- `README.md`: 项目说明
- `CHANGELOG.md`: 版本记录
- `.git/`: Git 版本控制历史
