Scaffold Logic

The proposal gives the review a health-side theory of individual sensory variability:

environmental stimuli -> sensory processing sensitivity -> context modulation -> information processing / learning

This connects directly with the medical coauthor pathway scaffold:

stimulus -> receptor -> pathway -> processing -> interpretation/action -> support

Together, they help frame wearable AI accessibility as support for heterogeneous sensory thresholds, sensory-cognitive processing, and context-dependent access needs.

How It Maps to CAWI

Proposal Concept Meaning for This Review CAWI Connection
Sensory processing sensitivity People differ in how strongly they perceive, tolerate, and respond to sensory stimuli. Sensory Modulation; Adaptive Transparency
Context modulation Sensory experience changes with attention, arousal, crowding, and environmental context. Contextual Perceivability; Attention Respect
Visual crowding Nearby visual elements can impair recognition and increase perceptual burden. Reduced visual density; AR overlay constraints
Physiological arousal Body state changes perception, learning, and tolerance for stimulation. Context-aware adaptation; inference accountability
Attention modulation Attention can amplify or gate sensory signals. Attention Respect; cognitive-state-aware prompting
Learning and predictive processing Sensory sensitivity may affect how people learn patterns and respond to expectations. Predictable feedback; cognitive legibility
