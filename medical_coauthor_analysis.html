Medical Coauthor Coding Results

Coding results from a medical coauthor interview on sensory pathways, receptors, thresholds, diagnosis/treatment logic, and how these concepts can strengthen the wearable AI accessibility scoping review.

Open Coding Memo Open Raw Transcript Search Protocol

Bottom Line

The interview helps reframe wearable AI accessibility as support for a sensory pathway, not just a set of UI preferences. The useful scaffold is:

stimulus -> receptor -> signal/transduction -> neural pathway -> processing -> interpretation/action -> support/treatment

This strengthens the review by giving CAWI a health-side foundation: wearable AI may sense external stimuli, modify stimulus intensity or channel, support perception when thresholds differ, reduce overload, compensate for missed signals, and help locate where an access breakdown occurs.

High-Level Codes

C1. Physiology vs Pathology

Start with normal sensory function, then examine atypical or pathological cases.

C2. Sensory Pathway as Architecture

Sensory access can be understood as receptor, transmission, processing, and output.

C3. Diagnosis as Debugging

Clinical diagnosis locates where the problem occurs; treatment intervenes there.

C4. Receptor as Key Concept

"Sensory" is too broad; the review needs receptor, pathway, and threshold language.

C5. Threshold and Sensitivity

Different bodies have different stimulus thresholds and response curves.

C6. No One-Size-Fits-All

The output should be an adaptive protocol, not a fixed universal setting.

C7. Modality-Specific Pathways

Visual, auditory, olfactory, tactile, thermal, pain, vibration, and proprioceptive pathways differ.

C8. Somatosensory Expansion

Touch, cold, pain, vibration, texture, and proprioception should be added to the plan.

C9. Reflex / Action Boundary

Some sensory pathways end in interpretation; others involve reflex, action, fatigue, or effectors.

C10. Wearable AI as Intervention

Wearable AI can support sensing, diagnosis, treatment, regulation, and accommodation.

C11. Dynamic Regulation

Sensitivity, overload, fatigue, and adaptation change over time and context.

C12. Term Ambiguity

Medical naming varies historically; search must include both functional and anatomical terms.

This layer should be piloted selectively because it can become very broad. Its purpose is conceptual grounding, not uncontrolled expansion.
